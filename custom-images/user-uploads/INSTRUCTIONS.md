# Sample Images for Testing

This folder is a convenient location to organize your custom images for the Connect the Dots game.

## How to Add Your Own Images

1. Copy your image files to this folder
2. Supported formats: JPG, PNG, GIF, WebP
3. Recommended size: 100x100 pixels
4. Maximum file size: 500KB per image

## Usage

1. Open `index.html` in your browser
2. Click "Customize Images"
3. Select "Use Custom Images" mode
4. Click "ğŸ“ Load from Folder (Batch Upload)"
5. In the file picker, navigate to this folder
6. Select all images you want to use (Ctrl+A or Cmd+A)
7. Click "Open" to upload
8. Click "Save & Apply"

**Note:** The game uses a standard browser file picker. You can select images from this folder or any other location on your computer.
