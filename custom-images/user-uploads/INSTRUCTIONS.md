# Sample Images for Testing

This folder contains sample images to help you test the batch upload feature.

## How to Add Your Own Images

1. Delete or replace these sample files with your own images
2. Supported formats: JPG, PNG, GIF, WebP
3. Recommended size: 100x100 pixels
4. Maximum file size: 500KB per image

## Usage

1. Open `index.html` in your browser
2. Click "Customize Images"
3. Click "Load from Folder (Batch Upload)"
4. Select all images from this folder
5. Click "Save & Apply"
